岗位名称：Java 后端开发实习生

岗位职责：
- 参与后端业务接口设计、开发和维护。
- 参与数据库表结构设计和基础 SQL 优化。
- 参与 AI 应用相关功能开发，包括大模型 API 调用、Prompt 优化和结果处理。
- 配合前端完成接口联调，保证系统稳定运行。

任职要求：
- 计算机相关专业本科及以上学历。
- 熟悉 Java 基础，了解集合、异常处理、IO、多线程等基础知识。
- 了解 Spring Boot 常用开发方式，熟悉 Controller、Service、Repository 分层。
- 熟悉 MySQL，理解表结构设计、索引和事务基础。
- 了解 HTTP、RESTful API、JSON 数据交互。
- 有 AI 应用项目、大模型 API 调用或完整 Web 项目经验优先。
