# CareerPilot 项目状态

更新日期：2026-05-24

## 当前定位

CareerPilot 是一个面向大学生求职场景的 AI 简历分析与模拟面试平台，目标是服务 Java 后端 / AI 应用后端实习方向。项目已经形成从简历制作、简历分析、模拟面试到历史记录管理的完整闭环。

## 已完成功能

### 1. 主工作台

- 三栏式 NotebookLM 风格布局。
- 左侧：简历上传/文本输入/岗位 JD。
- 中间：AI 分析结果。
- 右侧：模拟面试和 AI 对话。
- 页面整体固定高度，各区域独立滚动。

### 2. 简历上传与解析

- 支持 PDF、DOCX、TXT 文件上传。
- 使用 Apache Tika 提取简历文本。
- 支持直接粘贴简历文本。
- 支持文件大小和类型校验。

### 3. AI 简历分析

- 根据简历和岗位 JD 输出匹配分数。
- 输出分析摘要、优势、短板、优化建议、项目改写建议。
- 支持用户前端自定义 AI 配置。
- 未配置用户 AI 时使用后端默认模型。
- AI 异常时提供本地兜底结果。

### 4. 模拟面试

- 根据简历和 JD 生成模拟面试题。
- 支持重新生成。
- 面试题围绕 Java、Spring Boot、数据库、项目经验和 AI 应用展开。

### 5. AI 对话

- 支持右侧多轮 AI 对话。
- 对话携带当前简历和 JD 上下文。
- 必须使用用户前端自己的 AI 配置，避免消耗项目默认 Key。

### 6. 用户认证

- 手机号验证码注册。
- 用户名/手机号登录。
- 忘记密码和验证码重置。
- Redis 缓存验证码，Redis 不可用时本地缓存兜底。
- 轻量 JWT 鉴权。
- 密码加盐哈希保存。

### 7. 历史记录

- 分析完成后自动保存记录。
- 支持最近记录列表。
- 支持历史详情查看。
- 支持恢复历史现场。
- 支持删除记录。
- 登录用户按用户 ID 隔离记录。

### 8. 简历编辑器

- 8 套简历模板。
- A4 预览直接编辑。
- 支持模块标题直接编辑。
- 支持照片上传、显示/隐藏、证件照/方形头像切换。
- 支持主题色、字体、自定义主色。
- 支持缩放、适应宽度和 100% 查看。
- 支持导入当前简历文本。
- 支持导出 PDF、HTML、JSON、Markdown、WPS 可编辑 Word。
- 默认模板和其他模板已统一压缩留白，导出时尽量保留主题色。

### 9. 后端工程规范

- Controller / Service / Repository / DTO 分层。
- 统一返回结构。
- 全局异常处理。
- 参数校验。
- 文件校验。
- AI 调用超时和异常兜底。
- SLF4J 日志记录。
- 环境变量管理 AI 配置。
